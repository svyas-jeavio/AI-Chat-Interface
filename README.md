# AI Chat Interface

This project is a chat interface designed for AI applications. It supports both desktop and mobile UIs and includes the following features:

- **Authentication**: Login and logout functionality.
- **Dynamic Components**: Multiple components to support various types of assistant messages.
- **Responsive Design**: Optimized for mobile and desktop devices.

## Project Structure

The project is organized to ensure scalability and maintainability. Key directories and files include:

- `app/`: Contains the main application pages and components.
- `components/`: Houses reusable UI components, including those for assistant messages.
- `styles/`: Contains global and component-specific styles.
- `utils/`: Utility functions and helpers for the application.
- `services/`: API services and integrations.

For a detailed overview, explore the project structure in the repository.

## Getting Started

To run the project locally:

```bash
npm install
npm run dev
```
